# Quantitative Research

This portfolio emphasizes quantitative research frameworks that blend market data, sentiment, and statistical validation.

## Research pillars

- **Signal discovery:** find edges using alternative data and technical inputs
- **Validation:** use statistical tests and backtesting to confirm robustness
- **Interpretation:** explain model outcomes with finance context

## Sentiment and signal fusion

The `sentiment-analysis/` toolkit illustrates hybrid decision making:

- Technical indicators from price action
- News sentiment from RSS and web sources
- Social sentiment from Reddit communities
- Fundamentals and momentum overlays

## Research workflow

1. Fetch and preprocess multi-source data
2. Compute sentiment scores and technical metrics
3. Fuse signals into a unified hybrid score
4. Output ranked actionable signals with confidence levels

## Use cases

- **Pre-trade research:** generate watchlists and hypothesis tests
- **Strategy signals:** combine technical and sentiment signals for timing
- **Risk discovery:** detect sentiment-driven regime changes
